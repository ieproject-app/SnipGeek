# **App Name**: SnipBlog

## Core Features:

- MDX Content Loading: Dynamically load blog post content from local MDX files.
- Homepage Blog Listing: Display a list of blog posts on the homepage, loaded dynamically from MDX files.
- Single-Column Post Layout: Use a strict single-column layout for all blog posts and sections for optimal readability.
- Floating Pill Header: Implement a floating 'SG' text logo header that hides on scroll down and reappears on scroll up.
- Hero Image: Display a hero image at the top of each blog post.
- Post Metadata: Display post title, date, and other metadata in a readable format below the hero image.

## Style Guidelines:

- Primary color: Dark slate blue (#374151) to create a modern and sophisticated look.
- Background color: Very light gray (#F9FAFB), almost white, to ensure maximum contrast with the content.
- Accent color: Muted sky blue (#7DD3FC) to highlight interactive elements and CTAs.
- Body text: 'Inter', a grotesque sans-serif with a modern, machined, objective, neutral look.
- Headline: 'Space Grotesk', a proportional sans-serif with a computerized, techy, scientific feel. Code:'Source Code Pro'
- Centered single-column layout for all sections including homepage and individual blog posts.
- Smooth, subtle animations for the floating header appearance/disappearance.
