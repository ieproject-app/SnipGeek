Setelah beberapa kali mencoba, kami akhirnya menemukan solusi untuk masalah gaya pada blok kode yang tidak konsisten.

Masalahnya adalah selektor CSS yang terlalu spesifik atau konflik dengan gaya lain. Solusinya adalah dengan secara eksplisit mendefinisikan komponen untuk elemen `<pre>` di `mdx-components.tsx` dan menerapkan kelas Tailwind langsung di sana.

```tsx
// src/components/mdx-components.tsx

const MdxPre = ({ className, ...props }: React.HTMLAttributes<HTMLPreElement>) => (
    <pre
        className={cn(
            "rounded-lg p-6 my-6 overflow-x-auto",
            className
        )}
        {...props}
    />
);

export const mdxComponents: MDXComponents = {
    // ... other components
    pre: MdxPre,
}
```

Pendekatan ini memastikan bahwa semua blok kode akan selalu memiliki sudut membulat dan *padding* yang benar, tidak peduli apa pun.
