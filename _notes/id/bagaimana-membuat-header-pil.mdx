Header adalah elemen pertama yang dilihat pengunjung, dan untuk SnipGeek, saya ingin sesuatu yang terasa "ringan" tapi tetap fungsional. Akhirnya, saya memutuskan untuk menggunakan desain **Floating Pill Header**.

## Filosofi Desain

Meskipun saat ini kita sudah menerapkan standar *Precision Tech* dengan sudut **4px** pada seluruh kartu dan gambar di SnipGeek, header tetap saya biarkan berbentuk bulat sempurna (*full rounded*). Mengapa? Karena header berfungsi sebagai navigasi global yang harus terlihat kontras dan menonjol dari konten di bawahnya yang lebih kaku dan teknis.

## Detail Implementasi

Saya menggunakan kombinasi beberapa teknik CSS modern di Tailwind:
1. **Backdrop Blur**: Memberikan efek kaca transparan sehingga konten di bawahnya tetap terlihat samar.
2. **Sticky Positioning**: Memastikan navigasi selalu tersedia saat pengguna melakukan scroll.
3. **Z-Index Management**: Menempatkan header di lapisan paling atas agar tidak tertutup oleh elemen slider.

```tsx
// Cuplikan struktur utama header
<header className="fixed top-4 left-1/2 -translate-x-1/2 z-50">
  <nav className="bg-primary/90 backdrop-blur-sm rounded-full shadow-lg h-12">
    {/* Navigasi di sini */}
  </nav>
</header>
```
