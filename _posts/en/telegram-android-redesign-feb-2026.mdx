In early February 2026, Telegram for Android rolled out its most significant interface update since the app's inception on the platform. Version 12.4.x brings a comprehensive overhaul to the navigation structure, visual aesthetics, and several user experience (UX) elements.

This redesign adopts a visual approach called “Liquid Glass,” moving away from the classic side-menu structure to a more modern, transparent layout optimized for one-handed use. This approach was previously introduced on iOS and is now more consistently implemented on Android.

Here are the most prominent key changes in this update.

---

## 1. Key Visual Changes

### a. Bottom Navigation Bar

The most fundamental change is the relocation of the main navigation from the hamburger menu to a bottom navigation bar. This pattern aligns more closely with modern Android app design standards.

![Telegram's interface with the four main tabs at the bottom of the screen.](/images/articles/telegram-redesign/bottom-nav.webp)

The four main tabs are now permanently placed at the bottom of the screen:

*   Chats
*   Contacts
*   Settings
*   Profile

This bar remains visible as the user scrolls, allowing for faster menu switching without having to return to the main page.

---

### b. “Liquid Glass” Design

Telegram has adopted semi-transparent elements that provide a layered and more dynamic visual effect. This approach creates a modern feel, especially in light mode.

![Semi-transparent panels and a layered background in light mode.](/images/articles/telegram-redesign/liquid-glass.webp)

Key characteristics of this design include:

*   Panels with a light blur/transparency effect
*   More rounded element corners
*   A smoother visual hierarchy
*   A lighter and more spacious look

---

### c. Improved Search Access & Features

The search bar is now larger and permanently displayed at the top of the chat list, no longer hidden within a small icon.

![The new search bar above the chat list.](/images/articles/telegram-redesign/search-bar.webp)

Several features previously in the side menu have been moved to the overflow menu (three-dot icon) at the top right. This structure simplifies navigation while maintaining a clean main interface.

---

## 2. User Experience (UX) Enhancements

### a. Cross-Platform Consistency

This redesign aligns the user experience between Android and iOS. The navigation structure, visual style, and interactions now feel more consistent across both platforms.

### b. Performance and Responsiveness

According to the official release notes, the interface code was rebuilt from scratch. The results are:

*   Smoother transitions
*   Faster navigation response
*   Better stability across a wide range of Android devices

This approach demonstrates Telegram's focus on performance efficiency alongside visual modernization.

---

## 3. Additional Features in Version 12.4.x

### a. More Adaptive Group Management

This update also touches on community management aspects:

*   If a group owner leaves, ownership can be automatically transferred to another admin after a certain period.
*   Group owners can appoint a successor admin before leaving the group.

This feature improves the sustainability of community management without structural disruptions.

---

### b. Gift Crafting & Colored Bot Buttons

Telegram is introducing new interactive features:

**Gift Crafting**
Users can combine multiple digital gifts to create rarer or more exclusive items.

**Colored Bot Buttons and Emojis**
Bot developers can now customize buttons with colors and emojis, creating a more visual and intuitive experience for users.

This feature expands interaction flexibility, especially within the bot and digital monetization ecosystem.

---

## 4. Community Response

The update has sparked widespread discussion in the user community.

The majority have responded positively:

*   More ergonomic bottom navigation
*   A more modern look
*   More comfortable one-handed use

However, some long-time Android users have found the changes too close to iOS design styles.

This difference in response is normal, as large-scale changes often affect established usage habits.

---

## 5. Conclusion

The redesign of Telegram for Android in February 2026 is one of the most significant transformations in the app's history on the platform.

With:

*   More ergonomic bottom navigation
*   The “Liquid Glass” visual approach
*   Performance improvements
*   New management and interaction features

Telegram is making a strategic move to unify the cross-platform experience while updating its design standards on Android.

For Android users, this update brings a more modern, lightweight, and structured experience. To access these changes, ensure your Telegram app is updated via the Play Store.
