So, you're ready to perform a clean installation of Windows 11? Great choice! Using a USB installation media is the best way to get a fresh system free from manufacturer bloatware. And for this task, there's no better, lighter, or more user-friendly tool than **Rufus**.

This guide will walk you through the process step-by-step, from preparation to a ready-to-use USB, including the Rufus settings that matter most: **GPT vs MBR**, **UEFI vs Legacy**, Secure Boot, TPM checks, and what to do if the USB does not appear in the boot menu.

If your goal is a normal clean install on a modern laptop or PC, keep the default Windows 11 path: GPT, UEFI, and official installation media. The compatibility options are useful, but they should be used intentionally instead of being clicked blindly.

---

## #1 Key Preparation (Don't Skip!)

Before we begin, ensure you have the following ready to ensure a smooth process:

1.  **USB Flash Drive:** Prepare a flash drive with at least **8 GB** capacity. Make sure it's in good condition.
2.  **Windows 11 ISO File:** You'll need the Windows 11 installer file in `.iso` format.
    <DownloadButton id="windows-11-iso-24h2" />
3.  **Rufus App:** Download the latest version of Rufus. We've provided a safe link below.
    <DownloadButton id="rufus-installer" />
4.  **Backup Data:** This process will **erase all data** on the flash drive. Ensure you have backed up any important files from that drive.

<Callout variant="warning" title="Important Warning">
  The steps below will fully format your flash drive. All data inside will be permanently lost. Double-check your backups before proceeding.
</Callout>

## #2 Creating the Windows 11 Bootable Drive

![USB Flash Drive plugged into a laptop for Windows 11 installation](https://res.cloudinary.com/snipgeek/image/upload/q_auto/f_auto/v1781144529/snipgeek/images/_posts/rufus/rufus-banner.webp)

Once all preparations are complete, follow these steps:

<Steps>
  <Step>
    **Open Rufus & Select Device**
    Run the Rufus application you downloaded. Plug in your flash drive, and Rufus will automatically detect it in the **Device** section.

    ![Step 1: Selecting the USB device in Rufus interface](https://res.cloudinary.com/snipgeek/image/upload/q_auto/f_auto/v1781144533/snipgeek/images/_posts/rufus/rufus-4.9_1.png "Rufus Screenshot - Step 1")
    *Step 1: Ensure the correct flash drive is selected.*
  </Step>

  <Step>
    **Select Windows 11 ISO File**
    In the **Boot selection** area, click the **SELECT** button, then find and choose the Windows 11 `.iso` file you prepared earlier.

    ![Step 2: Selecting the Windows 11 ISO file in Rufus](https://res.cloudinary.com/snipgeek/image/upload/q_auto/f_auto/v1781144531/snipgeek/images/_posts/rufus/rufus-4.9_2.png "Rufus Screenshot - Step 2")
    *Step 2: Choosing the Windows 11 installer file.*
  </Step>

  <Step>
    **Configure Installation Options**
    After selecting the ISO, Rufus will automatically adjust several options. Keep the **Partition scheme** as **GPT** and **Target system** as **UEFI**, as these are the standards for Windows 11.
  </Step>

  <Step>
    **Start the Process**
    Once all settings are correct, click the **START** button at the bottom.
  </Step>

  <Step>
    **Windows Customization (Rufus Advanced Features)**
    A "Windows User Experience" pop-up will appear. This is where Rufus's magic happens. You can choose:
    *   **Remove requirement for 4GB+ RAM, Secure Boot and TPM 2.0:** Use this only when you fully understand the implications and for test environments where compatibility checks are necessary.
    *   Other options are optional, such as automatically creating a local account.

    ![Step 3: Customizing Windows options in Rufus](https://res.cloudinary.com/snipgeek/image/upload/q_auto/f_auto/v1781144537/snipgeek/images/_posts/rufus/rufus-4.9_3.png "Rufus Screenshot - Step 5")
    *Step 5: Review each option carefully and proceed only with settings that match your installation target.*

    Click **OK** when finished.
  </Step>

  <Step>
    **Wait for Completion**
    Rufus will begin creating the bootable disk. This process will take a few minutes. Wait until the progress bar turns green and the status shows **READY**.

    ![Step 4: Rufus status showing Ready after completion](https://res.cloudinary.com/snipgeek/image/upload/q_auto/f_auto/v1781144535/snipgeek/images/_posts/rufus/rufus-4.9_4.png "Rufus Screenshot - Step 6")
    *Once the status is READY, your flash drive is good to go!*
  </Step>
</Steps>

After it's finished, you can close the Rufus app. Your Windows 11 bootable flash drive is now ready to use for a clean installation on any computer that supports the UEFI system. Easy, right?

## #3 Recommended Rufus Settings for Windows 11

Use this table if you are unsure which option to choose in Rufus:

| Rufus option | Recommended setting | When to change it |
| --- | --- | --- |
| Device | Your target USB drive | Change only if more than one USB is plugged in. |
| Boot selection | Windows 11 ISO | Redownload the ISO if Rufus reports it as invalid. |
| Image option | Standard Windows installation | Use other modes only if you know the target scenario. |
| Partition scheme | GPT | Use MBR only for older Legacy/CSM-only systems. |
| Target system | UEFI | Legacy BIOS is not the normal Windows 11 path. |
| File system | NTFS or Rufus default | Let Rufus decide unless your firmware has a specific limitation. |

For most Windows 11 installs in 2026, **GPT + UEFI** is the safest baseline. It matches modern hardware and avoids many boot-menu problems.

## #4 Common Mistakes (and Fast Fixes)

Before you assume the USB failed, check these common issues:

1.  **USB not listed in BIOS Boot Menu**
  Recreate the USB and make sure **Partition scheme = GPT** for UEFI systems.

2.  **Boot starts but installer crashes early**
  Redownload the ISO and verify checksum if available. Corrupted ISO files are a frequent cause.

3.  **"This PC can't run Windows 11" appears**
  This usually means your hardware checks are active. Use Rufus compatibility options only for test/lab scenarios where you understand the trade-offs.

4.  **USB creation stops at a certain percentage**
  Try another USB port (preferably USB 2.0 for stability) and avoid running heavy background disk tasks.

5.  **The PC boots back to the old Windows installation**
  Open the one-time boot menu again and choose the USB entry that starts with `UEFI:`. Also check whether the USB drive is lower than the internal SSD in boot priority.

6.  **Rufus shows multiple USB devices**
  Unplug external drives you do not want to format. Rufus will erase the selected device, so do not continue until the target USB is obvious.

## #5 BIOS/UEFI Checklist Before Install

Use this quick checklist so installation starts on the first try:

*   Boot mode is set to **UEFI** (not Legacy/CSM).
*   Your USB is prioritized in the boot menu.
*   Secure Boot state matches the installation plan.
*   SATA mode (AHCI/RAID) is known before setup, especially on laptops.

These 4 checks prevent most "USB works on one PC but not another" cases.

## #6 What to Do After the USB Is Ready

Once Rufus shows **READY**, do not immediately wipe the target PC. Do these quick checks first:

1. Keep the USB plugged in and restart the target PC.
2. Open the boot menu key for your device, commonly <kbd>F12</kbd>, <kbd>F9</kbd>, <kbd>Esc</kbd>, or <kbd>F8</kbd> depending on the brand.
3. Choose the USB entry with the `UEFI:` label.
4. Stop at the Windows Setup screen and confirm the installer loads normally.
5. If you are doing a clean install, make sure your important files and license/account information are backed up first.

If you need a complete installation walkthrough after creating the USB, continue with the [clean install Windows 11 guide](/blog/clean-install-windows-11-step-by-step-guide). ## #7 Dual-Boot Safety Note (Windows + Linux)

If your target machine is dual-boot (for example Windows 11 + Ubuntu), avoid deleting partitions blindly during setup.

Safe approach:

1.  Identify the current Linux partitions first.
2.  Install Windows only to the intended partition/disk.
3.  Expect the bootloader to change after installation; plan to repair GRUB from Linux live USB if needed.

This keeps your dual-boot migration cleaner and reduces post-install recovery work.

## FAQ

### Is Rufus safe for creating a Windows 11 bootable USB?

Yes, Rufus is a widely used bootable USB tool. Use the official Rufus app and a Windows ISO you trust, then double-check the selected USB device before clicking **START**.

### Should I choose GPT or MBR for Windows 11?

Choose **GPT** for modern Windows 11 installations. MBR is mainly for older Legacy BIOS systems, while Windows 11 is designed around UEFI-era hardware.

### Why does my USB not appear in the boot menu?

The most common causes are wrong partition mode, Legacy/UEFI mismatch, a bad USB port, or the boot menu showing a non-UEFI USB entry. Recreate the USB with GPT + UEFI and try another port.

### Can Rufus bypass TPM or Secure Boot checks?

Rufus can show compatibility options during USB creation. Use them only when you understand the trade-offs, especially on unsupported hardware or test environments.

### Do I need internet during Windows 11 installation?

It depends on the Windows 11 edition and setup path. Having internet available is helpful for activation, account setup, and drivers, but prepare essential drivers separately if the target PC has unusual Wi-Fi or storage hardware.
