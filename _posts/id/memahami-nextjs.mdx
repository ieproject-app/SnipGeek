Next.js adalah sebuah framework React yang memberikan Anda pengalaman pengembangan terbaik dengan semua fitur yang Anda butuhkan untuk produksi: rendering hybrid statis & server, dukungan TypeScript, bundling cerdas, pre-fetching rute, dan banyak lagi. Tanpa perlu konfigurasi.

## Fondasi Dasar

Pada intinya, Next.js menyediakan cara terstruktur untuk membangun aplikasi React. Ini menangani banyak perkakas dan konfigurasi yang bisa membuat pusing saat memulai proyek baru dari awal.

<Note>
Ini adalah komponen catatan khusus! Anda bisa menggunakannya untuk menyorot informasi penting yang tidak boleh dilewatkan oleh pembaca. Sangat bagus untuk tips, pengingat, atau konteks tambahan.
</Note>

Berikut adalah contoh kode yang lebih panjang untuk mendemonstrasikan bagaimana `rehype-pretty-code` menangani pewarnaan sintaks dan luapan konten. Ini seharusnya sekarang dirender dengan tema `github-dark` dan dapat digulir secara horizontal di layar kecil.

```tsx
// src/app/layout.tsx
import { Inter } from 'next/font/google';
import './globals.css';
import { ThemeProvider } from '@/components/theme-provider';
import { Analytics } from '@vercel/analytics/react';
 
const inter = Inter({ subsets: ['latin'] });
 
export const metadata = {
  title: 'My Awesome App',
  description: 'Dibuat oleh create next app, dan sekarang memiliki deskripsi yang jauh lebih panjang untuk melihat bagaimana teks terbungkus dan bagaimana seluruh blok kode berperilaku ketika isinya sangat lebar.',
};
 
export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body className={inter.className}>
        <ThemeProvider attribute="class" defaultTheme="system" enableSystem>
          {children}
          <Analytics />
        </ThemeProvider>
      </body>
    </html>
  );
}
```

Contoh kode sebaris, seperti `const contoh = "halo"`, seharusnya memiliki latar belakang abu-abu sederhana dan bukan pewarnaan sintaksis penuh.

</Warning>
