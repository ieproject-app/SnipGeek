## Panduan Lengkap Membuat Bootable Windows 11 dengan Rufus

Melakukan instalasi bersih (clean install) Windows 11 merupakan langkah yang sangat direkomendasikan untuk mendapatkan sistem operasi yang stabil, ringan, dan bebas dari *bloatware* atau aplikasi bawaan yang tidak diperlukan. Metode paling populer dan efektif untuk melakukannya adalah dengan menggunakan media instalasi USB bootable.

Salah satu utilitas terbaik untuk tugas ini adalah **Rufus**—sebuah aplikasi portabel yang ringan, super cepat, dan sangat fleksibel. Keunggulan utamanya termasuk kemampuan untuk menyesuaikan instalasi, seperti melewati persyaratan sistem yang ketat (TPM 2.0 dan Secure Boot) yang sering menjadi kendala.

Panduan ini akan memandu Anda melalui seluruh proses secara sistematis, mulai dari persiapan hingga USB bootable Anda siap digunakan untuk instalasi.

## 1. Persiapan yang Diperlukan

Sebelum kita memulai, ada beberapa hal penting yang harus Anda siapkan untuk memastikan proses berjalan lancar.

### 1.1. USB Flash Drive
Gunakan flash drive dengan kapasitas minimal **8 GB**. Pastikan perangkat tersebut dalam kondisi baik dan tidak ada data penting di dalamnya.

### 1.2. File ISO Windows 11
Siapkan file installer Windows 11 dalam format `.iso`. Jika Anda belum memilikinya, Anda bisa mengunduhnya melalui tautan di bawah ini.

<DownloadLink id="windows-11-iso-24h2" />

### 1.3. Aplikasi Rufus
Gunakan versi terbaru Rufus untuk memastikan kompatibilitas dan fitur terkini.

<DownloadLink id="rufus-installer" />

### 1.4. Cadangkan Data Penting dari Flash Drive
> ⚠️ **Peringatan Penting**
> Proses ini akan **memformat total** flash drive Anda. Semua data, file, dan dokumen yang ada di dalamnya akan terhapus secara permanen. Pastikan Anda sudah memindahkan data penting ke lokasi lain sebelum melanjutkan.

## 2. Proses Pembuatan Bootable Windows 11

Setelah semua persiapan selesai, ikuti langkah-langkah berikut secara berurutan.

### Langkah 1 — Jalankan Rufus dan Pilih Perangkat
Buka aplikasi Rufus yang sudah Anda unduh. Hubungkan flash drive ke komputer Anda. Rufus akan secara otomatis mendeteksi perangkat tersebut dan menampilkannya di bagian **Device**.

![Rufus mendeteksi USB flash drive yang terhubung](/images/_posts/rufus/rufus-4.9_1.png)

Pastikan perangkat yang dipilih sudah benar untuk menghindari kesalahan format pada drive yang salah.

### Langkah 2 — Pilih File ISO Windows 11
Pada bagian **Boot selection**, klik tombol **SELECT**, lalu arahkan ke lokasi file ISO Windows 11 yang telah Anda siapkan.

![Memilih file ISO Windows 11 dari file explorer di dalam Rufus](/images/_posts/rufus/rufus-4.9_2.png)

Setelah file dipilih, Rufus akan secara otomatis menyesuaikan beberapa parameter lain berdasarkan file ISO tersebut.

### Langkah 3 — Verifikasi Skema Partisi dan Sistem Target
Untuk instalasi Windows 11 pada sistem modern, pastikan konfigurasi berikut sudah terpilih:
- **Partition scheme:** **GPT**
- **Target system:** **UEFI (non CSM)**

Konfigurasi ini adalah standar untuk hardware baru dan akan memastikan kompatibilitas terbaik.

### Langkah 4 — Mulai Proses
Klik tombol **START** untuk memulai proses pembuatan media bootable.

### Langkah 5 — Kustomisasi Pengalaman Instalasi Windows
Setelah menekan START, sebuah jendela konfigurasi tambahan bernama **Windows User Experience** akan muncul. Di sinilah letak keajaiban Rufus.

Anda dapat mencentang opsi berikut jika perangkat Anda tidak memenuhi persyaratan resmi Windows 11:
- **Remove requirement for 4GB+ RAM, Secure Boot and TPM 2.0**

![Jendela kustomisasi Windows di Rufus untuk melewati persyaratan sistem](/images/_posts/rufus/rufus-4.9_3.png)

Opsi ini sangat berguna untuk menginstal Windows 11 di komputer lama. Klik **OK** untuk melanjutkan.

### Langkah 6 — Tunggu Hingga Proses Selesai
Rufus akan mulai menyalin file-file instalasi ke flash drive. Proses ini biasanya membutuhkan beberapa menit, tergantung pada kecepatan flash drive dan port USB Anda.

![Status bar hijau di Rufus menunjukkan proses telah selesai dan siap](/images/_posts/rufus/rufus-4.9_4.png)

Ketika bilah status sudah penuh dan menampilkan tulisan **READY** dengan latar hijau, itu artinya flash drive Anda telah berhasil dibuat dan siap digunakan.

## Penutup
Selamat! Media instalasi Windows 11 Anda kini siap digunakan untuk melakukan instalasi bersih pada perangkat apa pun yang berbasis UEFI. Dengan menggunakan Rufus, Anda tidak hanya mendapatkan proses yang cepat tetapi juga fleksibilitas untuk mengatasi kendala persyaratan sistem.
