Punya monitor besar nganggur di meja tapi laptop masih jalan sendirian? Sayang banget. Nambah external monitor itu salah satu upgrade produktivitas paling simpel yang bisa kamu lakuin — nggak perlu beli laptop baru, cukup kabel dan sedikit setting.

Guide ini saya buat khusus buat kamu yang pakai **dual-boot Windows 11 dan Ubuntu**, karena caranya memang sedikit beda di tiap OS. Daripada cari dua tutorial terpisah, kita bahas sekalian di sini.

---

## Kenali Port di Laptopmu Dulu

Sebelum buru-buru colok kabel, pastiin dulu port apa yang tersedia di laptop kamu. Ini penting karena kabel atau adapter yang dibutuhkan bergantung dari sini.

{/* 📸 [GAMBAR DIBUTUHKAN] Foto atau ilustrasi berbagai jenis port laptop (HDMI, DisplayPort, USB-C, VGA) dengan label nama masing-masing. Idealnya satu frame dari sisi kiri/kanan laptop yang menunjukkan posisi port secara nyata. */}

Port yang paling umum ditemukan di laptop modern:

| Port | Keterangan |
|------|------------|
| **HDMI** | Paling umum, support video + audio sekaligus |
| **DisplayPort (DP)** | Kualitas lebih baik, umum di laptop gaming/workstation |
| **USB-C / Thunderbolt** | Port serbaguna, bisa output video kalau support DP Alt Mode |
| **VGA** | Sudah jadul, tapi masih ada di laptop lawas atau monitor lama |

Kalau port laptop dan monitor nggak sama — misalnya laptopmu cuma punya USB-C tapi monitornya HDMI — kamu butuh **adapter** atau **kabel converter**. Pastiin adapter yang kamu beli support video output, bukan sekadar USB hub biasa.

{/* 📸 [GAMBAR DIBUTUHKAN] Foto contoh adapter: USB-C to HDMI, Mini DisplayPort to HDMI, dll. */}

## Sambungkan Kabel ke Monitor

Langkah fisik ini berlaku sama baik di Windows maupun Ubuntu:

<Steps>
  <Step>

  ### Colokkan kabel
  Hubungkan kabel dari port laptop ke port input monitor.

  </Step>
  <Step>

  ### Pilih input source di monitor
  Nyalakan monitor, lalu tekan tombol **Input** atau **Source** di bagian bawah/samping monitor. Pastikan channel-nya sesuai port yang kamu colok — misalnya HDMI 1, HDMI 2, atau DisplayPort.

  </Step>
  <Step>

  ### Nyalakan atau bangunkan laptop
  Nyalakan laptop (atau bangunkan dari sleep). Di kebanyakan kasus, monitor langsung terdeteksi otomatis.

  </Step>
</Steps>

{/* 📸 [GAMBAR DIBUTUHKAN] Foto tangan sedang mencolokkan kabel HDMI ke sisi laptop — suasana meja kerja biar lebih relatable. */}

Kalau monitor belum muncul tampilannya setelah langkah di atas, lanjut ke konfigurasi sesuai OS kamu.

## Setting di Windows 11

### Pilih Display Mode

Setelah monitor terdeteksi, Windows 11 biasanya langsung muncul notifikasi di pojok kanan bawah. Atau kamu bisa akses langsung lewat shortcut:

<kbd>Windows</kbd> + <kbd>P</kbd>

{/* 📸 [SCREENSHOT DIBUTUHKAN] Popup "Project" Windows 11 yang muncul setelah tekan Win+P — menampilkan 4 opsi: PC screen only, Duplicate, Extend, Second screen only. */}

Ada 4 pilihan mode:

| Mode | Fungsi |
|------|--------|
| **PC screen only** | Hanya layar laptop yang aktif, monitor mati |
| **Duplicate** | Monitor menampilkan tampilan yang sama persis dengan laptop |
| **Extend** | Layar laptop + monitor jadi satu desktop lebar |
| **Second screen only** | Layar laptop mati, semua tampil di monitor eksternal |

Untuk kerja sehari-hari, **Extend** adalah pilihan paling produktif.

### Atur Resolusi dan Posisi Monitor

Kalau pakai mode Extend, perlu ngatur posisi monitor (kiri atau kanan layar laptop) supaya gerakan mouse-nya terasa natural.

<Steps>
  <Step>

  ### Buka Display Settings
  Klik kanan di desktop → **Display settings**, atau lewat **Settings → System → Display**.

  {/* 📸 [SCREENSHOT DIBUTUHKAN] Halaman Display Settings Windows 11 yang menunjukkan dua layar (kotak bertuliskan 1 dan 2), tombol "Identify", serta pengaturan Resolution dan Refresh Rate di bawahnya. */}

  </Step>
  <Step>

  ### Atur posisi monitor
  Drag kotak layar untuk atur posisi monitor relatif terhadap layar laptop. Klik **Identify** untuk tahu mana layar 1 dan layar 2.

  </Step>
  <Step>

  ### Set resolusi dan refresh rate
  Atur **Resolution** sesuai resolusi native monitor (biasanya 1920×1080 atau 2560×1440). Atur juga **Refresh rate** — kalau monitormu support 144Hz, pilih di sini.

  </Step>
  <Step>

  ### Terapkan perubahan
  Klik **Apply** → **Keep changes**.

  </Step>
</Steps>

### Set Monitor Utama (Main Display)

Kalau kamu mau taskbar dan desktop utama pindah ke monitor eksternal:

<Steps>
  <Step>

  ### Pilih layar utama
  Di Display Settings, klik kotak layar yang ingin dijadikan utama.

  </Step>
  <Step>

  ### Aktifkan sebagai main display
  Scroll ke bawah, centang **Make this my main display**.

  {/* 📸 [SCREENSHOT DIBUTUHKAN] Bagian bawah Display Settings Windows 11 menampilkan opsi "Make this my main display" dalam kondisi tercentang/highlighted. */}

  </Step>
</Steps>

## Setting di Ubuntu

Ubuntu dengan GNOME cukup pintar mendeteksi monitor eksternal secara otomatis. Tapi kalau perlu konfigurasi lebih lanjut, begini caranya.

### Via GUI — Settings → Displays

<Steps>
  <Step>

  ### Buka pengaturan display
  Buka **Settings** → **Displays**. Monitor yang terdeteksi akan muncul sebagai kotak-kotak layar.

  {/* 📸 [SCREENSHOT DIBUTUHKAN] Tampilan Settings → Displays di Ubuntu GNOME yang menunjukkan dua layar terdeteksi, dengan opsi Arrangement, Resolution, Refresh Rate, dan toggle On/Off per layar. */}

  </Step>
  <Step>

  ### Atur susunan dan resolusi
  Drag kotak layar untuk atur susunan/posisi monitor. Klik salah satu layar untuk ubah **Resolution** dan **Refresh Rate**-nya.

  </Step>
  <Step>

  ### Pilih Display Mode
  Di bagian atas halaman, pilih mode yang kamu inginkan:
  - **Mirror** — sama seperti Duplicate di Windows
  - **Join Displays** — sama seperti Extend
  - **Single Display** — hanya satu layar aktif

  </Step>
  <Step>

  ### Terapkan perubahan
  Klik **Apply**. Ubuntu bakal minta konfirmasi dalam 10 detik sebelum revert otomatis — klik **Keep Changes** kalau tampilannya sudah oke.

  </Step>
</Steps>

### Set Primary Display

Untuk menentukan layar mana yang jadi utama (tempat top bar dan dock muncul pertama kali):

- Klik layar yang ingin dijadikan primary
- Aktifkan toggle **Primary Display**

{/* 📸 [SCREENSHOT DIBUTUHKAN] Closeup toggle "Primary Display" di Settings → Displays Ubuntu dalam kondisi aktif. */}

### Via Terminal — xrandr (Opsional)

Buat yang lebih suka terminal, `xrandr` bisa jadi alternatif yang powerful.

<Steps>
  <Step>

  ### Lihat daftar output yang terdeteksi
  ```bash
  xrandr
  ```

  {/* 📸 [SCREENSHOT DIBUTUHKAN] Output perintah `xrandr` di terminal yang menampilkan daftar display output (eDP-1, HDMI-1, dll) beserta resolusi yang tersedia. */}

  Contoh output yang umum:
  ```
  eDP-1 connected primary 1920x1080+0+0
  HDMI-1 connected 1920x1080+1920+0
  ```

  </Step>
  <Step>

  ### Set posisi monitor
  Untuk set monitor HDMI di sebelah kanan layar laptop dengan resolusi 1920×1080:

  ```bash
  xrandr --output HDMI-1 --mode 1920x1080 --right-of eDP-1
  ```

  Atau kalau mau mirror:
  ```bash
  xrandr --output HDMI-1 --same-as eDP-1
  ```

  </Step>
</Steps>

<Callout variant="info" title="Nama output bisa berbeda">
  Nama output seperti `HDMI-1`, `eDP-1`, `DP-1` bisa beda-beda tergantung hardware laptopmu. Selalu sesuaikan dengan hasil `xrandr` di laptopmu sendiri.
</Callout>

## Troubleshooting — Monitor Nggak Terdeteksi?

Ini masalah yang paling sering bikin frustrasi. Tenang, ada beberapa hal yang bisa dicoba.

### Di Windows 11

<Steps>
  <Step>

  ### Coba deteksi manual
  Buka Display Settings → scroll ke bawah → klik tombol **Detect**.

  {/* 📸 [SCREENSHOT DIBUTUHKAN] Tombol "Detect" di bagian bawah Display Settings Windows 11 (ada di bawah tulisan "Multiple displays"). */}

  </Step>
  <Step>

  ### Update driver display
  Buka **Device Manager** → **Display adapters** → klik kanan GPU → **Update driver**. Atau download langsung dari situs resmi NVIDIA / AMD / Intel sesuai GPU kamu.

  </Step>
  <Step>

  ### Cek kabel dan input source monitor
  Pastiin kabel nggak longgar di kedua ujungnya. Cek tombol **Input/Source** di monitor dan pastiin channel-nya sesuai port yang kamu colok.

  </Step>
</Steps>

### Di Ubuntu

<Steps>
  <Step>

  ### Paksa deteksi ulang
  ```bash
  xrandr --auto
  ```

  </Step>
  <Step>

  ### Cek apakah monitor terdeteksi sistem
  ```bash
  xrandr --query
  ```

  Kalau monitor nggak muncul sama sekali di output `xrandr`, kemungkinan masalahnya ada di driver GPU atau kabel.

  </Step>
  <Step>

  ### Install driver GPU (khusus NVIDIA)
  Kalau kamu pakai laptop dengan GPU NVIDIA, pastiin driver proprietary sudah terinstall:

  ```bash
  ubuntu-drivers devices
  ```

  Lalu install driver yang direkomendasikan:
  ```bash
  sudo ubuntu-drivers autoinstall
  ```

  Setelah install, restart laptop lalu cek lagi.

  </Step>
  <Step>

  ### Restart display manager
  Kalau layar tiba-tiba blank setelah colok monitor:

  ```bash
  sudo systemctl restart gdm3
  ```

  </Step>
</Steps>

<Callout variant="tip" title="Tips Praktis">
  Kalau kamu sering gonta-ganti antara pakai monitor eksternal atau nggak, pertimbangkan bikin script `xrandr` kecil yang bisa dijalankan satu klik. Lumayan hemat waktu dibanding masuk Settings setiap saat.
</Callout>

## Penutup

Menghubungkan laptop ke external monitor itu sebenarnya simpel — baik di Windows 11 maupun Ubuntu prosesnya nggak beda jauh. Yang sering bikin ribet biasanya soal driver (terutama di Ubuntu dengan GPU NVIDIA) atau kabel/adapter yang nggak kompatibel.

Kalau setup dual monitor kamu sudah jalan, langkah berikutnya bisa explore **window management**: di Windows ada PowerToys FancyZones, di Ubuntu ada GNOME extension Pop Shell — keduanya bisa bikin workflow makin rapi dengan layout yang bisa dikustomisasi.
